A.题目名称：

Juggling Letters

题目大意：

n个字符串，每次操作可以选择两个字符串（可以相同）选择其中一个字符串中某一字符插入到另一字符串。问：操作任意次，能否使得所有字符串都相同？

思路：

统计n个字符串中26个小写字母的数量是否是n的倍数，是则YES，否则NO

代码：

#include<bits/stdc++.h>
using namespace std;
const int maxn=1e3+40;
int T,n,cnt[100];char s[maxn][maxn];
int main()
{
    cin>>T;
    while(T--)
    {
        cin>>n;
        for(int i=1;i<=n;i++)cin>>s[i]+1;
        memset(cnt,0,sizeof(cnt));
        for(int i=1;i<=n;i++)
        {
            int len=strlen(s[i]+1);
            for(int j=1;j<=len;j++)
            {
                cnt[s[i][j]-'a'+1]++;                
            }
        }
        int sign=0;
        for(int i=1;i<=26;i++)
        {
            if(cnt[i]==0)continue;
            if(cnt[i]%n)
            {
                sign=1;
                break;
            }
        }
        if(sign)printf("NO\n");
        else printf("YES\n");
    }
    return 0;
}

B.题目名称:

Power Sequence

题目大意:

n个数组成的数组，下标为0~n-1，首先对数组从小到大排序，然后有一操作：可以对任意数组中任意一个元素的值加一或者减一，代价为1；问最少需要多少代价可以使得数组a[i]=c^i.

思路:

首先将a[]从小到大排序,然后枚举c(1-MAX),再枚举到最小代价之前,答案都在不断减小,所以一旦不再减小就应该终止枚举.

代码:

#include<bits/stdc++.h>
using namespace std;
const int maxn=1e5+50;
typedef long long ll;
ll n,a[maxn],ans;
int check(ll x)
{
    ll sum=0,now=1;
    for(int i=1;i<=n;i++)
    {
        sum+=abs(a[i]-now);
        now*=x;
        if(sum>ans)return 0;
    }
    ans=sum;
    return 1;
}
int main()
{
    cin>>n;
    for(int i=1;i<=n;i++)cin>>a[i],ans+=a[i];
    sort(a+1,a+1+n);
    ans-=n;
    for(ll i=2;i<=1000000;i++)
    {
        if(!check(i))break;
    }    
    cout<<ans<<endl;
    return 0;
}

C.题目名称:

Multiples of Length

题目大意:

n个元素组成的数组，有一操作：对于某一段区间[l,r]，可以加上该区间长度len的任意倍数。题目说明：可以证明三次操作可以使得数组所有元素都变成0。

思路:

这是一个构造题:当n=1的时候,将[1,1]的数+1,再将[1,1]的数-1,最后[1,1]的数-a[1].当n>1的时候,将[1,1]的数改成n,即加上(n-a[1]),再将[2,n]的数全加上a[i] * (n-1).最后再-(a[i] * n).

代码:

#include<bits/stdc++.h>
using namespace std;
const int maxn=1e5+50;
typedef long long ll;
ll n,a[maxn],y[maxn],t[maxn];
int main()
{
    cin>>n;
    for(ll i=1;i<=n;i++)cin>>a[i];
    if(n==1)
    {
        printf("1 1\n1\n");
        printf("1 1\n-1\n");
        printf("1 1\n%d\n",-a[1]);
        return 0;
    }
    cout<<"1 1"<<endl;
    cout<<n-a[1]<<endl;
    cout<<"2 "<<n<<endl;
    t[1]=-n;
    for(int i=2;i<=n;i++)
    {
        y[i]=a[i]*(n-1);
        t[i]=-a[i]*n;
    }
    for(int i=2;i<=n;i++)cout<<y[i]<<' ';cout<<endl;
    cout<<"1 "<<n<<endl;
    for(int i=1;i<=n;i++)cout<<t[i]<<' ';cout<<endl;
    return 0;
}

D.题目名称:

Stoned Game

题目大意:

T和HL玩游戏,每人选择一个未被上一个玩家选择的堆中拿一个石头,一旦某一个人无法拿到石头,则另一个玩家获胜.

思路:

当一堆大于其余堆的总和的时候,T一定选择这个堆,则T获胜.
其余一般情况:假设最后一堆是x,则知道x的具体值,就知道了sum-x的奇偶性,若x=5,则当剩余9的时候,先手一定选择5这个堆,而后手再先手出手之前一定会阻止它,所以x=4.当x=4的时候,同理剩余7的时候也会出现上述情况,最后x=1,所以当sum为奇数T获胜,否则HL获胜.

代码:

#include<bits/stdc++.h>
using namespace std;
const int maxn=1000+50;
int T,n,a[maxn];
int main()
{
    cin>>T;
    while(T--)
    {
        cin>>n;
        int sum=0,maxx=0;
        for(int i=1;i<=n;i++)cin>>a[i],sum+=a[i],maxx=max(maxx,a[i]);
        if(maxx>sum-maxx)printf("T\n");
        else 
        {
            if(sum%2==1)printf("T\n");
            else printf("HL\n");
        }
    }
    return 0;
}
