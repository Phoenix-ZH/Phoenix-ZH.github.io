1.题目：

CodeForces - 371C A-Hamburgers

题目大意：

做汉堡包，需要的材料用字符串给出了。

已有的材料数在第二行。（顺序：B,S,C）

商店卖的材料的需要的钱数在第三行。（顺序：B,S,C）

第四行是你有的钱数。

问最多做多少汉堡。

思路：

看到（1<=r<=10^12)和“最多”就觉得是二分。二分汉堡的数量，已知已经拥有的汉堡B，S，C材料的数量，也可以得到需要的总数量，由此知道需要购买的数量。然后计算总钱数，就可以进行判断了。
ps：WA了两次：
第一次：注意ans1，ans2，ans3，可能是负的，一旦为负就要定为0，不然结果偏大。
第二次：注意最多数量可能比10^12更大，所以右边界要大一点。

代码：

#include<bits/stdc++.h>
using namespace std;
const int maxn=10000+10;
long long n1,n2,n3,h1,h2,h3,m1,m2,m3,r;
bool check(long long mid)
{
    long long N1=n1*mid-h1,N2=n2*mid-h2,N3=n3*mid-h3;
    long long ans1=N1*m1,ans2=N2*m2,ans3=N3*m3;
    if(ans1<0)ans1=0;
    if(ans2<0)ans2=0;
    if(ans3<0)ans3=0;
    long long ans=ans1+ans2+ans3;
    if(ans<=r)return 1;
    return 0;
}
int main()
{
    string s;
    cin>>s;
    cin>>h1>>h2>>h3>>m1>>m2>>m3>>r;
    for(int i=0;i<s.length();i++)
    {
        if(s[i]=='B')n1++;
        else if(s[i]=='S')n2++;
        else n3++;
    }
    long long l=0,ans=0,r=1100000000000;
    while(l<=r)
    {
        long long mid=(l+r)/2;
        if(check(mid))
        {
            l=mid+1;
            ans=mid;
        }
        else 
        {
            r=mid-1;
        }
    }
    cout<<ans<<endl;
    return 0;
}

2.题目：

CodeForces - 343C B-Read Time

题目大意：

给你n个探头，m个要读的轨道，n个探头的初始位置是h1—-hn（从小到大），m个轨道的位置为p1———pn(也是从小到大)，探头可以左移或右移，这些探头可以一起动，每移动一格的时间为1，探头读轨道不计时间，如果要读的轨道上就有探头那么就不需要时间，找最小的时间来读完这些轨道。

思路：

二分需要的时间。判断条件：
从左到右，对每一个探头进行操作：
1.如果该探头左边没有待读取的区域，则将探头移到右边可达的最远处；
2.如果该探头左边有带读取区域，这就需要分两种情况：
1）先左移到带读取区域，再右移到可达的最远处；
2）先右移到最远的区域（ps：这个区域需要让探头能够到左边的带读取区域）
ps：如果探头到左边待读取区域的时间大于了所给时间，直接返回0；
3.然后就得到了右边可达的最远处，这样在它左边的所有区域都是已读取的区域
4.如果所有区域可读取则返回1，否则返回0；

代码：

#include<bits/stdc++.h>
using namespace std;
const long long maxn=100000+10;
long long n,m,h[maxn],p[maxn],vis[maxn];
bool check(long long mid)
{
    memset(vis,0,sizeof(vis));
    long long opt=1;
    for(long long i=1;i<=n;i++)
    {
        long long r;
        if(p[opt]>=h[i])
        {
            r=h[i]+mid;
        }
        else 
        {
            r=max(p[opt]+(mid-(h[i]-p[opt])),(mid+p[opt]-h[i])/2+h[i]);
            if(h[i]-p[opt]>mid)return 0;
        }
        while(opt<=m&&p[opt]<=r)
        {
            vis[opt]=1;
            opt++;
        }
        if(opt==m+1)break;
    }
    for(long long i=1;i<=m;i++)
        if(!vis[i])
            return 0;
    return 1;
}
int main()
{
    cin>>n>>m;
    for(long long i=1;i<=n;i++)cin>>h[i];
    for(long long i=1;i<=m;i++)cin>>p[i];
    long long l=0,r=21000000000,ans=0;
    while(l<=r)
    {
        long long mid=(l+r)/2;
        if(check(mid))
        {
            ans=mid;
            r=mid-1;
        }
        else
        {
            l=mid+1;
        }
    }
    cout<<ans<<endl;
    return 0;
}
