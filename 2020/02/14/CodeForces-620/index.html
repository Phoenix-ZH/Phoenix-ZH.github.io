A：题目名称：

Professor GukiZ’s Robot

题目大意：

机器人很好玩

一开始在(x1,y1)

最后在(x2,y2)

每秒钟内横坐标最多变化1（也可以不变化）纵坐标也是

问最少几秒钟到 x1 和 x2 绝对值在10^9以内 。

题解：

取两者坐标x，y相差的最大值，开longlong保平安。

代码：

#include<cstdio>
#include<cstring>
#include<iostream>
#include<algorithm>
using namespace std;
int main()
{
    long long x1,x2,y1,y2;
    scanf("%lld%lld%lld%lld",&x1,&y1,&x2,&y2);
    long long maxx=max(abs(x1-x2),abs(y1-y2));
    printf("%lld\n",maxx);
    return 0;
}

B:题目名称：

Grandfather Dovlet’s calculator

题目大意：

从0-9每一个数字由若干个小木棍组成，给定a，b，问搭出a-b的每一个数字需要多少个小木棍。

题解：

t[i]表示数字i需要的木棍(i∈[0,9]),然后枚举。

代码：

#include<cstdio>
#include<cstring>
#include<iostream>
#include<algorithm>
using namespace std;
long long a,b,t[10]={6,2,5,5,4,5,6,3,7,6};
int main()
{
    long long ans=0;
    scanf("%lld%lld",&a,&b);
    for(long long i=a;i<=b;i++)
    {
        long long now=i;
        while(now)
        {
            long long x=now%10;
            now/=10;
            ans+=t[x];
        }
    }
    printf("%lld\n",ans);
    return 0;
}

C：题目名称：

Pearls in a Row

题目大意：

现在有N个数，你的任务是将这N个数尽可能切割成多段。每一段必须包括两个相同的数。

输出的第一行为尽可能切割的最大段数K。

接下来K行，每行为两个整数lj, rj (1 ≤lj ≤ rj ≤ n) ，表示切割的区间范围

如果存在多个合法的切割方法，输出任意一个即可。

如果不能切割成合法的情况，输出”-1”.

数据第一行为N(1 ≤ N ≤ 3·105) 。

数据的第二行包括N个数ai(1 ≤ a[i] ≤ 109) 。

题解：

看到N<=300000，然后再看到a[i]的数据，由于答案只与二者是否相等有关，那么它的值就无关紧要了，所以直接离散化。离散化之后就是如何分割了，首先就想到了贪心，一旦又符合条件的就直接分割，如何考虑这个分案的合理性呢？假设区间 [x,y]为一段，a[x]==a[y],如果w∈(x,y),z>y,那么我们也必须选择[x,y] ,否则区间(x,w)不符合题意。如果我们遇到有两个相等了而不分割，这样并不能使数量增加，所以每遇到两个相等就立即分割，这样受到影响的只有最后一段，因为最后一段可能无法成立，那它直接与前一段合并，即前一段的l[opt]=n就可以了。

代码：

#include<cstdio>
#include<cstring>
#include<iostream>
#include<algorithm>
using namespace std;
const long long maxn=300000+5;
long long n,a[maxn],b[maxn],vis[maxn],opt,l[maxn],r[maxn];
int main()
{
    scanf("%lld",&n);
    for(long long i=1;i<=n;i++)
    {
        scanf("%lld",&a[i]); 
        b[i]=a[i];
    }
    sort(b+1,b+1+n);
    long long tot=unique(b+1,b+1+n)-b-1;    
    for(long long i=1;i<=n;i++)
    {
        a[i]=lower_bound(b+1,b+1+tot,a[i])-b;
    }
    for(long long i=1;i<=n;i++)
    {
        if(vis[a[i]]==opt+1)
        {
            vis[a[i]]=0;
            opt++;
            l[opt]=r[opt-1]+1;
            r[opt]=i;
        }
        else
        {
            vis[a[i]]=opt+1;
        }
    }
    if(opt==0)
    {
        printf("-1\n");
        return 0;
    }
    if(r[opt]!=n)
    r[opt]=n;
    printf("%lld\n",opt);
    for(long long i=1;i<=opt;i++)
    {
        printf("%lld %lld\n",l[i],r[i]);
    }
    return 0;
}

ps：一开始一直WA，原因使if，else那段写反了，注意每一个vis[a[i]]都需要修改！
