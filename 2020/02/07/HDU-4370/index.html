题目名称：

HDU - 4370 0 or 1

题目链接：

https://vjudge.net/problem/HDU-4370#author=cjyl

题目大意：

给出一个n×n的矩阵，求一个符合题意的n×n的只含有0和1的矩阵，使得两个矩阵位置对应的元素相乘后和最小。

题意要求01矩阵：

  1. X12+X13+…+X1n=1

    2.X14+X24+…+X(n-1)n=1

    3.∑Xki (1<=k<=n)=∑Xij (1<=j<=n).（1<i<n）

题解：

对于条件1，可以说明1结点一定有且仅有一个出度，并且不是指向自己。
对于条件2，可以说明n结点一定有且仅有一个入度，并且该入度的发出者不是自己。
对于条件3，可以说明每个2-n-1的每一个结点的出度等于入度。
而题目要求求ΣCij * X ij最小值，即求最短路。
但是要分2种情况：
原因：结点1一定会指向结点i(i∈[2,n])，然后结点i一定指向别的结点，注意结点i不可以直接指向它自己，因为这样出度就不等于入读。最终会指向结点n，或者指向结点1形成自环。对于结点n来讲也是一样。

情况1：结点1最终指向结点n，则dist[n]就是最小值

情况2：结点1形成自环，那么结点n也必须形成自环。则dist[1]+dist[n]为答案。
针对情况2需要跑两次spfa，第一次spfa以1为起点，第二次spfa以n为起点。

最终答案为min(ans,d1+d2)其中ans=dist[n],d1=dist[1]，d2=dist[n]（这是第二次跑的spfa的结果）

代码：

#include<cstdio>
#include<cstring>
#include<iostream>
#include<algorithm>
#include<queue>
using namespace std;
const int maxn=30000+5;
int n,c[305][305],head[maxn],vis[maxn],dist[maxn];
struct node
{
    int to,next,w;
}edge[maxn*2];
void spfa(int s)
{
    queue<int>q;
    memset(dist,0x3f3f3f3f,sizeof(dist));
    memset(vis,0,sizeof(vis));
    for(int i=1;i<=n;i++)
    {
        if(i==s)
        continue;
        else 
        {
            dist[i]=c[s][i];
            q.push(i);
            vis[i]=1;
        }
    }
    while(!q.empty())
    {
        int x=q.front();q.pop();vis[x]=0;
        for(int i=1;i<=n;i++)
        {
            if(dist[i]>dist[x]+c[x][i])
            {
                dist[i]=dist[x]+c[x][i];
                if(!vis[i])
                {
                    vis[i]=1;
                    q.push(i);
                }
            }
        }
    }
}
int main()
{
    while(scanf("%d",&n)!=EOF)
    {
        for(int i=1;i<=n;i++)
        {
            for(int j=1;j<=n;j++)
            {
                scanf("%d",&c[i][j]);
            }
        }
        spfa(1);
        int ans=dist[n];
        int d1=dist[1];
        spfa(n);
        int d2=dist[n];
        printf("%d\n",min(ans,d1+d2));        
    }
}
