题目名称：

POJ - 3281 Dining or P2891 [USACO07OPEN]吃饭Dining

题目链接：

https://vjudge.net/problem/POJ-3281

题目大意：

有F种食物和D种饮料，每种食物或饮料只能供一头牛享用，且每头牛只享用一种食物和一种饮料。
现在有n头牛，每头牛都有自己喜欢的食物种类列表和饮料种类列表，问最多能使几头牛同时享用到自己喜欢的食物和饮料。（1 <= f <= 100, 1 <= d <= 100, 1 <= n <= 100）

题解：

我一开始想到的办法稍微有点复杂。首先着很明显是最大流问题，难点就只有建图，
最开始我想到的建图是一个超级源点，指向牛，然后牛指向食物，由于食物只有一次使用机会，所以食物–>食物’（流量为1）表示的是，
该食物的数量为1，然后食物’指向牛’，牛’指向饮料，饮料–>饮料’（流量为1），然后饮料’指向超级汇点。但是这种方法有问题。
至于问题，我现在没找到，之后再想想。

ps:后面想通了，由于牛–>食物，食物–>食物’，食物’–>牛’；这样单次经过的牛可能不是同一头牛，会导致错乱，当然答案不对。

正确做法：显然上面的建图有点复杂了，对于牛来讲，它必须满足食物和饮料均有，所以将牛放在中间，超级源点指向牛，而每种牛只有一条，
所以牛–>牛’（容量为1），牛’–>饮料，饮料指向超级源点，跑网络流dinic就可以了

#include<cstdio>

using namespace std;
const int maxn=300000+5;
const int N=101;
int n,k=1,head[maxn],vis[maxn],deep[maxn],f,d,s,t,ans;
struct node
{
    int to,next,w;
}edge[maxn*2];
void add(int u,int v,int w)
{
    edge[++k].to=v;edge[k].next=head[u];head[u]=k;edge[k].w=w;
}
int bfs()
{
    queue<int>q;
    memset(deep,0,sizeof(deep));
    deep[s]=1;
    q.push(s);
    while(!q.empty())
    {
        int x=q.front();q.pop();
        for(int i=head[x];i;i=edge[i].next)
        {
            if(!deep[edge[i].to]&&edge[i].w)
            {
                deep[edge[i].to]=deep[x]+1;
                q.push(edge[i].to);
            }
        }
    }
    return deep[t];
}
int dfs(int x,int F)
{
    if(x==t)return F;
    int f=0;
    for(int i=head[x];i;i=edge[i].next)
    {
        if(deep[edge[i].to]==deep[x]+1&&edge[i].w)
        {
            int w=dfs(edge[i].to,min(F,edge[i].w));
            edge[i].w-=w;edge[i^1].w+=w;f+=w;F-=w;
        }
    }
    if(!f)deep[x]=0;
    return f;
}
int dinic()
{
    while(bfs())
    {
        ans+=dfs(s,0x3f3f3f3f);
    }
    return ans;
}
