题目名称：

POJ - 3660 Cow Contest

题目链接：

https://vjudge.net/problem/POJ-3660#author=arasHi_

题目大意：

有n头牛，m条比赛结果，每行两个数字，第一个数字表示的是胜利者。
问有几头牛的排名可以确定？

题解：

排名可以确定，这该如何理解？对于结点i，如果箭头指向i所包含的结点数为ans1，i箭头指出所包含的结点数为ans2，若ans1+ans2==n-1就说明结点i的排名可以确定。问题在于如何确定ans1，ans2的值。
举个例子：若a可以到b，b可以到c，那么a就可以到c。这就是传递闭包了。

如何进行传递闭包呢？以上例子，a可以到b，b可以到c，则dist[a][b]=1,dist[b][c]=1,则a可以到c，即dist[a][c]=1；用floyd进行收缩即可。然后每个结点是否可以到达剩余的n-1个结点，或者剩余n-1个结点能否到达该结点，即ans1+ans2==n。判断条件就是dist[i][j]==1||dist[j][i]==1;

代码：

#include<cstdio>
#include<cstring>
#include<iostream>
#include<algorithm>
using namespace std;
const int maxn=100+5;
int n,m,dist[maxn][maxn];
int main()
{
    scanf("%d%d",&n,&m);
    for(int i=1;i<=m;i++)
    {
        int a,b;scanf("%d%d",&a,&b);
        dist[a][b]=1;
    }
    for(int k=1;k<=n;k++)
    {
        for(int i=1;i<=n;i++)
        {
            for(int j=1;j<=n;j++)
            {
                if(dist[i][k]==1&&dist[k][j]==1)
                dist[i][j]=1;
            }
        }
    }
    int ans=0;
    for(int i=1;i<=n;i++)
    {
        int sign=0;
        for(int j=1;j<=n;j++)
        {
            if(i==j)continue;
            if(dist[i][j]||dist[j][i])continue;
            else sign=1;
        }
        if(sign==0)ans++;
    }
    printf("%d\n",ans);
    return 0;
}
