A 题目名称：

Divisibility Problem

题目大意：

给定正整数 a,b，求至少把 a加上多少才能让 a为 b的倍数。

多组数据，T≤10^4，a,b≤10^9。

题解：

如果a%b==0，说明a就是b的倍数，那么答案为0；
否则找到最近的a使得a%b==0，显然a=((a/b)+1)*b;答案就是a‘-a；

代码：

#include<bits/stdc++.h>
using namespace std;
long long a,b,T;
int main()
{
    cin>>T;
    while(T--)
    {
        cin>>a>>b;
        if(a%b==0)printf("0\n");
        else 
        {
            long long x=a/b+1;
            long long ans=b*x;
            long long tot=ans-a;
            cout<<tot<<endl;
        }
    }
    return 0;
}

B 题目名称：

K-th Beautiful String

题目大意：

给定正整数 n,k，考虑所有由 n-2个 a 和 2 个 b 构成的字符串，输出其中字典序第 k小的。

题解：

找规律：设b1，b2，b1更靠左。当b1在第二个位置上(从右开始)，共有1，当b1在第三个位置上,共有2个，当b1在第x个位置上，共有x-1个。由此就可以找到规律了。可以通过排名确定b1的位置，然后k-之前的所有数，就是b2所处的位置。

如何确定b1的位置呢？
假设b1处在第x位，b2处在第1位，那么在这之前的所有数量就是1+2+…+(x-2),即
(x-1)(x-2)/2;设tot=x-1.那么就要满足tot (tot-1)/2<=k.解方程即可。那么x=tot+1，b2的位置就是k-(x-1)*(x-2)/2;

代码：

#include<bits/stdc++.h>
using namespace std;
long long T,n,k;
int main()
{
    cin>>T;
    while(T--)
    {
        cin>>n>>k;
        if(k==1)
        {
            for(long long i=1;i<=n-2;i++)printf("a");
            printf("bb\n");
            continue;
        }
        double delta=1+sqrt(1+8*k);
        long long tot=delta/2.0;
        //while(tot*(tot-1)/2<k)tot++;
        //tot--;
        long long x1=tot+1,y1=k-tot*(tot-1)/2;
        if(tot*(tot-1)/2==k)
        {
            x1--;y1=k-(tot-1)*(tot-2)/2;
        }
        for(long long i=n;i>=1;i--)
        {
            if(i!=x1&&i!=y1)printf("a");
            else printf("b");
        }
        cout<<endl;
    }
    return 0;
}

C 题目名称：

Ternary XOR

题目大意：

有T组数据，每组数据给一个n，表示数字长度为n， 要求 a,b 的异或值为 x且 max(a,b) 尽可能的小 .

题解：

若x[i]==0，则a[i]=b[i]=0;若x[i]=1,如果此情况第一次出现，那么随便让a[i],b[i]其中一个为1都可以，这里假设a[i]=1；如果这不是第一次出现，那么都让b[i]=1。因为之前a数组的高位已经比b更大了，所以a数组是最大的，但是要让a尽量小，所以后面就将数字让给b。如果x[i]==2,如果此前a，b数组的大小还未确定，那就让a[i]=b[i]=1,如果a数组已经更大了，就让b[i]=2.

代码：

#include<bits/stdc++.h>
using namespace std;
const int maxn=50000+5;
int T,n,len,a[maxn],b[maxn];char s[maxn];
int main()
{
    cin>>T;
    while(T--)
    {
        memset(a,0,sizeof(a));
        memset(b,0,sizeof(b));
        cin>>len;
        int sign=0;
        cin>>s+1;
        for(int i=1;i<=len;i++)
        {
            int x;x=s[i]-'0';
            if(x==1&&sign==0)
            {
                a[i]=1;b[i]=0;sign=1;
            }
            else if(x==1&&sign==1)
            {
                a[i]=0;b[i]=1;
            }
            else if(x==2&&sign==1)
            {
                a[i]=0;b[i]=2;
            }
            else if(x==2&&sign==0)
            {
                a[i]=b[i]=1;
            }
            else 
            a[i]=b[i]=0;
        }
        for(int i=1;i<=len;i++)cout<<a[i];
        cout<<endl;
        for(int i=1;i<=len;i++)cout<<b[i];
        cout<<endl;
    }
    return 0;
}

D 题目名称：

Carousel

题目大意：

有q组询问,每组询问如下:

已知一个有n(3≤n≤2*10^5)个点的环,点i的类型为ai,现在需要给每个点进行染色,要求相邻不同类型的点的颜色不同且所用颜色数最小.输出颜色数及一种染色方案即可.(颜色从1开始)

题解：

先线性跑一遍，设定a[1]=1,然后1，2，1，2…交替出现，若判断数组a都是同一个数字，说明一种颜色就够了，然后全部输出1；否则就说明，至少需要两种情况，第n个数字，由于与a[n-1]和a[1]相接，这时候就有四种情况：

1.当a[n]!=a[1]&&a[n]!=a[n-1],如果c[n-1]==c[1]，那么c[n]只要不与它们相同就好了，如果c[n-1]!=c[1]，这时候需要知道中间是否存在有相邻的数字相同(这个在之前的线性运行中就可以得到结果)，由前面可以得到最后一对相等数字对应的下标，从它开始所以颜色1与2对换，那么c[n-1]就可以与c[1]相同了。如果都不行，就说明至少要三种颜色了， c[n]=3；

2.a[n]==a[n-1]&&a[n]!=a[1],只需要两种颜色，c[n]不与c[1]相同即可。

3.a[n]!=a[n-1]&&a[n]==a[1]，只需要两种颜色，c[n]不与c[n-1]相同即可。

4.a[n]==a[n-1]&&a[n]==a[1]，只需要两种颜色，c[n]两种都可。

代码：

#include<cstdio>
#include<cstring>
#include<iostream>
#include<algorithm>
using namespace std;
const int maxn=200000+5;
int T,n,a[maxn],c[maxn]; 
int main()
{
    cin>>T;
    while(T--)
    {
        cin>>n;
        for(int i=1;i<=n;i++)cin>>a[i];
        int sign=1,e=0;c[1]=1;
        for(int i=2;i<=n;i++)
        {
            if(c[i-1]==1)c[i]=2;
            else c[i]=1;
            if(a[i]==a[i-1])e=i-1;
            else sign=2;
        }
        if(sign==1)
        {
            cout<<'1'<<endl;
            for(int i=1;i<=n;i++)cout<<'1'<<' ';
            cout<<endl;
            continue;
        }
        if(a[n]!=a[n-1]&&a[n]!=a[1])
        {
            if(c[n-1]==c[1])
            {
                if(c[1]==1)c[n]=2;
                else c[n]=1;
            }
            else if(e)
            {
                c[e+1]=3-c[e+1];
                for(int i=e+2;i<=n-1;i++)c[i]=3-c[i];
                if(c[n-1]==c[1])
                {
                    if(c[1]==1)c[n]=2;
                    else c[n]=1;
                }
            }
            else 
            {
                sign=3;c[n]=3;
            }
        }
        else if(a[n]==a[n-1]&&a[n]!=a[1])
        {
            if(c[1]==1)c[n]=2;
            else c[n]=1;
        }
        else if(a[n]!=a[n-1]&&a[n]==a[1])
        {
            if(c[n-1]==1)c[n]=2;
            else c[n]=1;
        }
        else if(a[n]==a[n-1]&&a[n]==a[1])c[n]=1;
        cout<<sign<<endl;
        for(int i=1;i<=n;i++)cout<<c[i]<<' ';
        cout<<endl;
    }
    return 0;
}

ps：考试的时候一直WA，原因是：没有考虑到 a[n]与a[n-1],最开始的for 只枚举到了n-1.

E 题目名称：

Tree Queries

题目大意：

给你一个以 1为根的有根树.
每回询问 k节点v1,v2⋯vk
求出是否有一条以根节点为一端的链使得询问的每个节点到此链的距离均≤1.
只需输出可行性, 无需输出方案.

题解：

这道题真的是血亏啊。被傻逼D搞死了，这道题都没看，简单的LCA就这么放弃了。
每次询问给了k个结点，找到深度最深的，显然这个结点与1结点构成了链，然后其余的点与此点找到LCA，看深度差是否小于等于1.

代码：

#include<bits/stdc++.h>
using namespace std;
const int maxn=200000+5;
int n,m,k=1,head[maxn],deep[maxn],fa[maxn][25],a[maxn];
struct node
{
    int to,next;
}edge[maxn*2];
void add(int u,int v)
{
    edge[++k].to=v;edge[k].next=head[u];head[u]=k;
}
void dfs(int u,int f)
{
    fa[u][0]=f;
    for(int i=head[u];i;i=edge[i].next)
    {
        if(edge[i].to==f)continue;
        deep[edge[i].to]=deep[u]+1;
        dfs(edge[i].to,u);
    }
}
int LCA(int x,int y)
{
    if(deep[x]<deep[y])swap(x,y);
    int depth=deep[x]-deep[y];
    for(int i=20;i>=0;i--)
    if(depth&(1<<i))
    x=fa[x][i];
    if(x==y)return x;
    for(int i=20;i>=0;i--)
        if(fa[x][i]!=fa[y][i])
        {
            x=fa[x][i];y=fa[y][i];
        }
    return fa[x][0];
}
int main()
{
    cin>>n>>m;
    for(int i=1;i<n;i++)
    {
        int a,b;
        cin>>a>>b;
        add(a,b);add(b,a);
    }
    deep[1]=1;
    dfs(1,1);
    for(int i=1;i<=20;i++)
        for(int j=1;j<=n;j++)
            fa[j][i]=fa[fa[j][i-1]][i-1];
    for(int i=1;i<=m;i++)
    {
        int w;cin>>w;int opt=0,maxx=0;
        for(int j=1;j<=w;j++)
        {
            cin>>a[j];
            if(maxx<deep[a[j]])
            {
                maxx=deep[a[j]];
                opt=a[j];
            }
        }
        int sign=0;
        for(int j=1;j<=w;j++)
        {
            if(a[j]==opt)continue;
            int f=LCA(a[j],opt);
            if(abs(deep[f]-deep[a[j]])>1)
            {
                sign=1;
                break;
            }
        }
        if(sign==0)printf("YES\n");
        else printf("NO\n");
    }
    return 0;
}

F 题目名称：

Make k Equal

题目大意：

给你一个数组及其长度 n 以及一个值 k
仅通过将 最小元素自增/最大元素自减 的方式, 以使得至少有 k个元素相等, 输出所需的最少步数.

题解：

首先先将数组a从小到大排序。如果数组中已经有k个数字大小相等了，那么直接输出0即可。对于ai，要让数组a有k-1个数字和它一样大，就有3种方法。
1.全部修改ai左边的数字：
如果ai左边(包括ai)共有k个数字，那么ans=a[i] k-sum[i]；
如果ai左边(包括ai)超过k个数字，那么ans=a[i] i-sum[i]-(i-k);
如果ai左边(包括ai)不足k个数字，自然就与条件矛盾了。
ps：需要主要每次是修改最小的值，所以修改其左边的数字，那么最终全部都要为ai或者ai-1.

2.全部修改ai右边的数字：
如果ai右边(包括ai)共有k个数字，那么ans=-a[i] k+(sum[n]-sum[i-1]);
如果ai右边(包括ai)超过k个数字，那么ans=-a[i] i+(sum[n]-sum[i-1])-(n-i+1-k);
如果ai右边(包括ai)不足k个数字，自然就与条件矛盾了。
ps：需要主要每次是修改最大的值，所以修改其右边的数字，那么最终全部都要为ai或者ai+1.

3.两边都要修改：
那么左边要么为ai，要么为ai-1；右边要么为ai，要么为ai+1；所以

最终取最小值即可。

代码：

#include<bits/stdc++.h>
using namespace std;
const long long maxn=200000+5;
long long n,k,a[maxn],sum[maxn],cnt[maxn];
int main()
{
    cin>>n>>k;
    for(long long i=1;i<=n;i++)cin>>a[i];
    sort(a+1,a+1+n);
    for(long long i=1;i<=n;i++)
    {
        if(a[i]==a[i-1])cnt[i]=cnt[i-1]+1;
        else cnt[i]=1;
        if(cnt[i]>=k)
        {
            printf("0\n");
            return 0;    
        }
    }
    for(long long i=1;i<=n;i++)sum[i]=sum[i-1]+a[i];
    long long ans=1e18;
    for(long long i=1;i<=n;i++)
    {
        long long ans1=1e18,ans2=1e18,ans3=1e18;
        if(i>=k)
        {
            ans1=a[i]*i-sum[i]-(i-k);
        }
        if(n-i+1>=k)
        {
            ans2=sum[n]-sum[i-1]-(n-i+1)*a[i]-(n-i+1-k);
        }
        ans3=a[i]*i-sum[i]+sum[n]-sum[i-1]-(n-i+1)*a[i]-(n-k);
        ans=min(ans,min(min(ans1,ans2),ans3));
    }
    cout<<ans<<endl;
    return 0;
}
