A 题目名称：

EhAb AnD gCd

题目大意

给你一个数x，输出一组 (a,b)使得 a和 b的最大公约数和最小公倍数的和为 x，若有多种答案，输出任意一种即可。
第一行输入一个 t，有 t组数据，之后每组数据输入一个数 x。
对于每组数据，输出两个用一个空格隔开的数 a,b。若有多种答案，输出任意一种即可。

题解

1是任意一个数的公约数，任意数是它和1的倍数，那么x=1+y。1，y就是答案

代码

#include<cstdio>
#include<cstring>
#include<iostream>
#include<algorithm>
int n;int T;
int main()
{
    scanf("%d",&T);
    while(T--)
    {
        scanf("%d",&n);
        if(n==2)
        {
            printf("1 1\n");
        }
        else 
        {
            printf("1 %d\n",n-1);
        }
    }
    return 0;
}

B 题目名称：

CopyCopyCopyCopyCopy

题目大意

共有 t 组数据，每组数据给你一个长为 n(1≤n≤105) 的序列(1≤ai≤109)，将它复制 n遍，请在这个复制后的序列里找一个最长上升子序列，输出它的长度。

题解

由于可以重复n遍，可以依次按序取每组中最小的那个，依次增大，那么一旦有重复的，数量就要-1，所以只需要求数组中有多少数字重复即可。去重求总数即可。

代码

#include<cstdio>
#include<cstring>
#include<iostream>
#include<algorithm>
using namespace std;
const int maxn=100000+5;
int T,a[maxn],n,vis[maxn],b[maxn];
int main()
{
    scanf("%d",&T);
    while(T--)
    {
        scanf("%d",&n);
        for(int i=1;i<=n;i++)
        {
            scanf("%d",&a[i]);
            b[i]=a[i];
        }
        sort(b+1,b+1+n);
        int tot=unique(b+1,b+1+n)-b-1;
        printf("%d\n",tot);
    }
    return 0;
}

C 题目名称：

Ehab and Path-etic MEXs

题目大意

给定一个 n个节点 n-1 条边的树
要求给边重新标注边权
分别为 0,1,2…n-2
然后使得树上任意两点u v的MEX(u,v) 的最大值最小。
MEX(u,v) 是 u到 v的简单路径没有出现的自然数中最小的数。

题解

这道题具体到单点上进行分析，如果这是一条链，每一个点最多连两条边。如图：

这样来看，1-n之间没有出现的最小数只能是n-1,所以边权怎么定义都无法影响MEX了。所以这种情况随便赋值即可。

如果一个点连接了三个点。如图：


三条边(分别是0，1，2)0恰好将图分成两个部分，u，v要么处于一个部分，要么处于两个不同部分。如处于相同部分就是0（但如果u、v之前存在0边权的边，则最小值是1）.如果处于两个不同部分，就是除了相邻的两条边之外的最小的值。这样MEX的最大值就是2，明显是最优的。

如果一个点连接超过三个点，方法如上图，实际上只需要定义三边就够了。
ps:若总点数小于等于3，就特判一下。

代码

#include<cstdio>
#include<cstring>
#include<iostream>
#include<algorithm>
using namespace std;
const int maxn=100000+5;
int n,ind[maxn],k=1,head[maxn],ans[maxn];
struct node
{
    int to,next;
}edge[maxn*2];
void add(int u,int v)
{
    edge[++k].to=v;edge[k].next=head[u];head[u]=k;
}
int main()
{
    scanf("%d",&n);
    if(n==2)
    {
        printf("0\n");
        return 0;
    }
    if(n==3)
    {
        printf("0\n1\n");
        return 0;
    }
    int now=1;
    for(int i=1;i<=n-1;i++)
    {
        int u,v;
        scanf("%d%d",&u,&v);
        add(u,v);add(v,u);
        ind[u]++;ind[v]++; 
        if(ind[u]==3)
        {
            for(int i=head[u];i;i=edge[i].next)
            {
                ans[i/2]=now;
                now++;
            }
        }
        else if(ind[v]==3)
        {
            for(int i=head[v];i;i=edge[i].next)
            {
                ans[i/2]=now;
                now++;
            }
        }
        if(ind[u]==3|ind[v]==3)
        {
            for(int i=1;i<=n-1;i++)
            {
                if(!ans[i])
                {
                    ans[i]=now;
                    now++;
                }
                else continue;
            }    
            break;
        }
    }
    if(now==1)
    {
        for(int i=1;i<=n-1;i++)
        {
            ans[i]=now;
            now++;
        }
    }
    for(int i=1;i<=n-1;i++)
    printf("%d\n",ans[i]-1);
    return 0;
}

D 题目名称：

Ehab the Xorcist

题目大意

给两个整数 u，v，求出长度最短的数组，使得数组内元素的按位异或为 u，元素之和为 v。输入一行，两个数 u，v(u,v≤10^18)。
如果有解，输出两行，第一行输出一个数n，第二行输出n个数，表示数组里的元素。如果有多个答案，输出任意一组。
如果无解，输出仅一行，输出”-1”。

题解

先讨论一下特殊情况：
如果u和v都等于0，显然答案是0
如果u==v，只需要u自己就可以了。
如果u>v,异或的值都大于总和了，显然不可能。
其余情况：
已知y^y==0,那么不妨让x+y+y=v。如果(v-x)%2==1，说明u,v不是同为奇数或者同为偶数，这种情况无法得到答案，直接输出-1.如果(v-x)%2==0,则y=(v-x)/2,那么x^y^y==x，那么答案就是三个数字：x y y。但是这种情况可能不是最好的。有些情况其实两个数字就够了。不妨设v=(x+y)+y.要满足(x+y)^y==u,由于x^y^y==u,所以
(x+y)^y==x^y^y,即x+y==x^y;两个数异或等于这两个数相加，显然x&y==0.即当x&y==0时，答案就是两个数字 x+y y。

代码

#include<cstdio>
#include<cstring>
#include<iostream>
#include<algorithm>
using namespace std;
long long n,m;
int main()
{
    scanf("%lld%lld",&n,&m);
    if(n==0&&m==0)
    {
        printf("0\n");
        return 0;
    }
    if(n==m)
    {
        printf("1\n%lld\n",n);
        return 0;
    }
    if(n>m)
    {
        printf("-1\n");
        return 0;
    }
    long long yu=m-n;
    if(yu%2==0)
    {
        long long x=n,y=yu/2;
        if((x&y)==0)
        {
            printf("2\n%lld %lld\n",x+y,y);
        }
        else 
        {
            printf("3\n%lld %lld %lld\n",x,y,y);
        }
    }
    else 
    {
        printf("-1\n");
        return 0;
    }
}
