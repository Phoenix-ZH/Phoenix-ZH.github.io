A：题目名称：

Yet Another Tetris Problem

题目大意：

第一行一个t，表示有t组数据。每组数据的第一行是一个数n，接下来一行是一个长度为n的数组a。 分别表示每一列的方块的高度。
现在有两个操作：
1.当至少一个ai>0时，选择一些i(1<=i<=n)，将ai加2.
2.当所有的ai>0时，将所有的ai减1。
问最后可不可以将所有的ai都变成0. 如果能，输出”YES”(没有引号)，不然输出”NO”(没有引号)

题解：

问题转化过来就是，可以在任意一列上+2，问最后可不可以等高。那就看highmax与其余各列的高度差是否都为偶数即可。

代码：

#include<cstdio>
#include<cstring>
#include<iostream>
#include<algorithm>
using namespace std;
const int maxn=1000+5;
int T,n,a[maxn];
int main()
{
    scanf("%d",&T);
    while(T--)
    {
        int maxx=0;
        scanf("%d",&n);
        for(int i=1;i<=n;i++)
        {
            scanf("%d",&a[i]);
            maxx=max(maxx,a[i]);
        }
        int sign=0;
        for(int i=1;i<=n;i++)
        {
            int t=maxx-a[i];
            if(t%2==1)
            {
                sign=1;
                break;
            }
        }
        if(sign==0)printf("YES\n");
        else printf("NO\n");
    }
    return 0;
}

B: 题目名称：

Yet Another Palindrome Problem

题目大意：

给定一个长度为 n 的序列 a，问其中是否有长度至少为 3 的子序列 b 为「回文序列」，如果有，则输出 YES，否则输出 NO。

注意：本题有多组数据，对每一组数据，都要给出相应答案。

题解：

只要满足回文串的长度为3即可。那就找到任意相等的两个数字，看他们之间间隔是否大于1。对于每一个类型的数字，记录一下最先出现的位置，用vis[]表示。然后倒着枚举每一个数字，看它的位置与同样大小数字最先出现的位置相差是否大于1，一旦符合就YES；如果找不到就NO。

代码：

#include<cstdio>
#include<cstring>
#include<iostream>
#include<algorithm>
using namespace std;
const int maxn=5000+5;
int T,n,a[maxn],vis[maxn],b[maxn];
int main()
{
    scanf("%d",&T);
    while(T--)
    {
        scanf("%d",&n);
        memset(vis,0,sizeof(vis));
        memset(b,0,sizeof(b));
        for(int i=1;i<=n;i++)
        {
            scanf("%d",&a[i]);
            if(vis[a[i]])
                b[i]=vis[a[i]];
            else vis[a[i]]=i;
        }
        int sign=0;
        for(int i=n;i>=1;i--)
        {
            if(i-b[i]>=2&&b[i]!=0)
            {
                printf("YES\n");
                sign=1;
                break;
            }
        }
        if(sign==0)printf("NO\n");
    }    
    return 0;
}

C: 题目名称：

Frog Jumps

题目大意：

有T组数据。每组数据是一串由L与R组成的字符串。青蛙从0出发，到n+1停止。n就是字符串长度(n不会给出)。若青蛙停在ai上，ai==‘L’就只能向左跳，ai==’R’就只能向右跳。求出最小的d，使得每一次青蛙跳的距离不超过d并且可以到达n+1.

题解：

假如ai==‘L’，意味着青蛙要向左跳。由于最终要跳到n+1，就必须尽量落在ai==’R’上，假设ai—>aj,ai==’R‘，aj==’L’，如果j-i==d，如果中间没有R了，那显然不能满足它跳到n+1去，若有R那肯定跳到最后一个ak==’R’ 上。如果j-i<d,就看后面是否有ak==’R’(注意范围),或者i—j之间是否有ak==’R’.总之这样推断后，发现每一次都要落在ai==’R’上才能到达n+1处。现在就是如何求d了，对d进行二分即可。

代码：

#include<cstdio>
#include<cstring>
#include<iostream>
#include<algorithm>
using namespace std;
const int maxn=200000+10;
int T,len;char s[maxn];
bool check(int mid)
{
    int opt=0,now=0,sign=0;
    while(opt<=len+1)
    {
        if(s[opt]=='R'&&opt-now<=mid)
        {
            now=opt;
        }
        else if(opt-now>mid)
        {
            sign=1;break;
        }
        opt++;
    }
    if(sign==1)return 0;
    else return 1;
}
int main()
{
    scanf("%d",&T);
    while(T--)
    {
        scanf("%s",s+1);
        len=strlen(s+1);
        s[len+1]='R';
        int l=0,r=200000+5,ans=0;
        while(l<=r)
        {
            int mid=(l+r)/2;
            if(check(mid))
            {
                ans=mid;
                r=mid-1;
            }
            else 
            {
                l=mid+1;
            }
        }
        printf("%d\n",ans);
    }
    return 0;
}

D: 题目名称：

Pair of Topics

Ps：本场比赛WA了4次，把自己坑得人都没了QAQ

题目大意：

给定n组ai，bi。求有多少组满足ai+aj>bi+bj(i<j)

题解：

移项之后就是求 (ai-bi)+(aj-bj)>0有多少组。设ci=ai-bi.就是求ci+cj>0有多少组。将c数组从小到大排好。minn为最靠后的小于或等于0的数，maxx为最靠前的大于或等于0的数。这里特别要注意：如果存在0，那么minn=maxx=i(a[i]==0)，如果存在多个0，那么minn=maxx=i(i是最后一个0所在位置)，那么大于0的个数就是n-maxx+1.注意：如果maxx==n，可能a[maxx]=0或者a[maxx]>0；也可能minn=maxx=1，意味着可能a[minn]=a[maxx]=0,也可能是全为负数。这里的特殊情况较多，很容易WA。(主要还是设计算法时候没有注意到），然后将minn倒着向前推，如果c[minn]+c[maxx]>0,ans+=n-maxx+1(意味着maxx后面的也都满足)，如果c[minn]+c[maxx]<=0，就maxx++，直到c[minn]+c[maxx]>0(注意maxx不可以超过n),所以还要判断是否c[minn]+c[maxx]>0。
ps：minn=maxx=1时候，也可能全正，直接输出答案即可。ans=(n-1)*n/2;

代码：

#include<cstdio>
#include<cstring>
#include<iostream>
#include<algorithm>
using namespace std;
const long long maxn=200000+10;
long long n,a[maxn],b[maxn],c[maxn];
long long cmp(long long x,long long y)
{
    return x<y;
}
int main()
{
    scanf("%lld",&n);
    for(long long i=1;i<=n;i++)scanf("%lld",&a[i]);
    for(long long i=1;i<=n;i++)scanf("%lld",&b[i]);
    for(long long i=1;i<=n;i++)c[i]=a[i]-b[i];
    sort(c+1,c+1+n,cmp);
    long long minn=1,maxx=1;
    for(long long i=1;i<=n;i++)
    {
        if(c[i]==0)
        {
            minn=maxx=i;
        }
        if(c[i]>0&&c[i-1]<0)
        {
            minn=i-1;
            maxx=i;
        }
        if(c[i]>0&&c[i-1]>0)break;
    }
    long long ans=0;
    if(minn==maxx)
    {
        while(c[maxx]<=0&&maxx<=n-1)maxx++;
    }
    ans=(n-maxx+1)*(n-maxx)/2;
    if(maxx==1)
    {
        printf("%lld\n",ans);
        return 0;
    }
    for(long long i=minn;i>=1;i--)
    {
        if(c[i]+c[maxx]>0)
        {
            ans+=n-maxx+1;
        }
        else 
        {
            while(c[i]+c[maxx]<=0&&maxx<=n-1)maxx++;
            if(c[i]+c[maxx]>0)ans+=n-maxx+1;
        }
    }
    printf("%lld\n",ans);
    return 0;
}

E: 题目名称：

Sleeping Schedule

题目大意：

Vova 可以选择在ai小时或者ai-1小时后睡觉，每次睡h个小时，如果开始睡觉那一刻位于l和r之间就是符合要求的。问一共有多少个符合要求。

题解：

每一次睡觉时有两个选择：ai个小时后睡或者ai-1个小时后睡觉，但是由于之前i-1次也会出现这样的选择。如果单纯直接递推，就无法判断枚举的上次睡觉时间是否可以到达。所以vis[i][j]记录第i次在j时刻睡觉。dp[i][j]表示第i次在j时刻睡觉所累积的符合要求的数量。显然dp[1][a[i]]=1,dp[1][a[i]-1]=1.vis[1][a[1]]=1,vis[1][a[1]-1]=1,然后开始枚举ai，和ai-1次睡觉的时刻。必须满足vis[i-1][j]=1，即上一次睡觉可以在j时刻睡觉才可以继续。那么第i次睡觉可以在(j+a[i])%h或者(j+a[i]-1)%h开始。对vis数组进行记录。然后更新dp数组(看睡觉时刻是否处于l和r之间).然后再dp[n][j]中求maxx(0<=j<=h-1).
ps:l<=a[1]<=r时候，dp[1][a[1]]=1;l<=a[1]-1<=r时候，dp[1][a[i]-1]=1,而vis数组直接标记即可。

代码：

#include<cstdio>
#include<cstring>
#include<iostream>
#include<algorithm>
using namespace std;
const int maxn=3000+5;
int n,h,l,r,dp[maxn][maxn],a[maxn],vis[maxn][maxn];
int main()
{
    scanf("%d%d%d%d",&n,&h,&l,&r);
    for(int i=1;i<=n;i++)scanf("%d",&a[i]);
    if(a[1]>=l&&a[1]<=r)
    {
        dp[1][a[1]]=1;
    }
    vis[1][a[1]]=1;
    if(a[1]-1>=l&&a[1]-1<=r)
    {
        dp[1][a[1]-1]=1;
    }
    vis[1][a[1]-1]=1;
    for(int i=2;i<=n;i++)
    {
        for(int j=0;j<=h-1;j++)
        {
            if(!vis[i-1][j])continue;
            int x=(j+a[i])%h;
            vis[i][x]=1;
            if(x>=l&&x<=r)dp[i][x]=max(dp[i][x],dp[i-1][j]+1);
            else dp[i][x]=max(dp[i][x],dp[i-1][j]);
            x=(j+a[i]-1)%h;
            vis[i][x]=1;
            if(x>=l&&x<=r)dp[i][x]=max(dp[i][x],dp[i-1][j]+1);
            else dp[i][x]=max(dp[i][x],dp[i-1][j]);
        }
    }
    int ans=0;
    for(int j=0;j<=h-1;j++)
    ans=max(ans,dp[n][j]);
    printf("%d\n",ans);
    return 0;
}

F: 题目名称：

Sleeping Schedule

题目大意：

给定一棵 n 个节点无根树，每个节点 u有一个颜色au，若 au 为 0 则 u 是黑点，若 au为 1 则 u是白点。
对于每个节点 u，选出一个包含 u的连通子图，设子图中白点个数为 cnt1，黑点个数为 cnt2，请最大化 cnt1 - cnt2。并输出这个值。

题解：

默认1为根，从头到底进行一边dfs，每次进入dfs，先更新w1，w2数组，若为白色则w1[i]=1否则w2[i]=1,则sum[i]=w1[i]-w2i.然后从底部进行回溯：若edge[i].to(即子树的头节点)它的sum[edge[i].to]>0,那么sum[u]+=sum[edge[i].to].这样只是更新了头节点。而子树与其头节点的关系并没有更新。假如一个子树，它的父节点所在子树的sum[u]>0,而该节点所在子树的sum[edge[i].to]>0,那么显然父亲节点的子树一定包含了该子节点所在的子树，那么sum[edge[i].to]=sum[u].如果sum[edge[i].to]<0，就说明该节点为黑色。sum[edge[i].to]=sum[edge[i].to]+sum[u];然后继续dfs。

代码：

#include<cstdio>
#include<cstring>
#include<iostream>
#include<algorithm>
using namespace std;
const int maxn=200000+5;
int n,v[maxn],k=1,w1[maxn],w2[maxn],sum[maxn],son[maxn],head[maxn];
struct node
{
    int to,next;
}edge[maxn*2];
void add(int u,int v)
{
    edge[++k].to=v;edge[k].next=head[u];head[u]=k;
}
void dfs(int u,int f)
{
    if(v[u]==0)w2[u]=1;
    if(v[u]==1)w1[u]=1;
    sum[u]=w1[u]-w2[u];
    for(int i=head[u];i;i=edge[i].next)
    {
        if(edge[i].to==f)continue;
        dfs(edge[i].to,u);
        if(sum[edge[i].to]>0)
        sum[u]+=sum[edge[i].to];
    }
}
void dfs2(int u,int f)
{
    for(int i=head[u];i;i=edge[i].next)
    {
        if(edge[i].to==f)continue;
        if(sum[edge[i].to]>=0&&sum[edge[i].to]<sum[u])
        sum[edge[i].to]=sum[u];
        else if(sum[edge[i].to]<0&&sum[u]>=0)
        sum[edge[i].to]+=sum[u];
        dfs2(edge[i].to,u);
    }
}
int main()
{
    scanf("%d",&n);
    for(int i=1;i<=n;i++)scanf("%d",&v[i]);
    for(int i=1;i<n;i++)
    {
        int u,v;
        scanf("%d%d",&u,&v);
        add(u,v);add(v,u);
    }
    dfs(1,1);
    dfs2(1,1);
    for(int i=1;i<=n;i++)printf("%d ",sum[i]);cout<<endl;
    return 0;
}
