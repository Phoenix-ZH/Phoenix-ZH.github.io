2-SAT概念介绍

提问方式：

有n个变量x[i]，x[i]的取值为0/1.

给定m个约束条件：若x[i]=i,则x[j]=j；若x[i]=~i,则x[j]=~j.

判断是否存在一组变量的合法赋值，满足上述约束条件。

解决方法：

1.建立一个2N个点的有向图，i和~i分别设定为i和i+N；

2.对于每个约束条件可以建边：i->j;同时它的逆否命题也可以连边:(~j)->(~i)（原命题和逆否命题等价）

3.Tarjan算法跑上述的图，求处有向图的SCC

4.如果对于i,x[i]=x[i+N]即i和~i属于同一个连通块，必然无解。即对于任意i,x[i]!=x[i+N]是答案有解的充要条件

2-SAT训练题目

1. AcWing 370. 卡图难题

思路

问是否存在一组合法赋值，使得m组：a&b=c,a|b=c,a^b=c都成立.

1)a&b=0 —> a->~b,b->~a
2)a&b=1 —> ~a->a,~b->b(由于~a和a不可以同时出现在一个连通块中，而要使得a&b=1, a和b都必须为1，所以只能选择a=1，b=1,~a和~b不可以被选中，那么让~a->a,~b->b,一旦选择~a或者~b，那么a或者b就被选了，就可能存在a与~a同时存在一个连通块中，或者b与~b同时存在一个连通块中，并且要让原命题和逆否命题同时满足且a，b同时选中，只能让~a->a,~b->b)
3)a|b=0: —> a->~a,b->~b(和上述类似，最后只能选~a和~b,并且a和~a,b和~b不可以在同一个连通块中)
4)a|b=1: —> ~b->a,~a->b
5)a^b=0: —> a->b,~a->~b,b->a,~b->~a(a^b=0，说明a与b始终相同，如果a=1，那么b=1，所以a->b,如果a=0,那么b=0，所以~a->~b，其余同理)
6)a^b=1: —> ~a->b,a->~b,b->~a,~b->a(与上述类似，但是a与b始终不同)

假如选择了”->”左边的，那么一定就会推出”->”右边的，但是如果选择了”->”右边的，不一定推出”->”左边的

根据上述的结论进行建边，i与~i分别用i与i+n进行表示，然后跑tarjan，判断是否所有的点都满足color[i]!=color[i+n]，（a不可能同时等于0又等于1，所以color[i]与color[i+n]不可以在一个连通块中）

代码

#include<bits/stdc++.h>
using namespace std;
const int maxn=1e6+50;
struct Edge{
    int to,next;
}edge[maxn*2];
int k=1,head[maxn];
void add(int u,int v){
    edge[++k].to=v;edge[k].next=head[u];head[u]=k;
} 
int dfn[maxn],low[maxn],tot,sta[maxn],Num[maxn],vis[maxn],color[maxn];
void tarjan(int x){
    low[x]=dfn[x]=++tot;sta[++sta[0]]=x;vis[x]=1;
    for(int i=head[x];i;i=edge[i].next){
        if(!dfn[edge[i].to]){
            tarjan(edge[i].to);
            low[x]=min(low[x],low[edge[i].to]);
        }
        else if(vis[edge[i].to]){
            low[x]=min(low[x],dfn[edge[i].to]);
        }
    }
    if(low[x]==dfn[x]){
        Num[0]++;
        do{
            color[sta[sta[0]]]=Num[0];
            vis[sta[sta[0]--]]=0;
            Num[Num[0]]++;
        }while(sta[sta[0]+1]!=x);
    }
}
int n,m;
int main(){
    cin>>n>>m;
    for(int i=1;i<=m;i++){
        int a,b,c;string s;
        cin>>a>>b>>c>>s;
        if(s[0]=='A'){
            if(c==0){
                add(a,b+n);
                add(b,a+n);
            }
            else{
                add(a+n,a);
                add(b+n,b);
            }
        }
        else if(s[0]=='O'){
            if(c==0){
                add(a,a+n);
                add(b,b+n);
            }
            else{
                add(a+n,b);
                add(b+n,a);
            }
        }
        else{
            if(c==0){
                add(a,b);add(b,a);
                add(b+n,a+n);add(a+n,b+n);
            }
            else{
                add(a,b+n);add(b,a+n);
                add(b+n,a);add(a+n,b);
            }
        }
    }
    for(int i=1;i<=n;i++){
        if(!dfn[i]){
            tarjan(i);
        }
    }
    int sign=0;
    for(int i=1;i<=n;i++){
        if(color[i]==color[i+n]){
            sign=1;
            break;
        }
    }
    if(sign)cout<<"NO"<<endl;
    else cout<<"YES"<<endl;
}

2.[HNOI2010] 平面图判定

思路

根据题意，对于一个无向图G=(V,E)需要判断它是否为平面图，并且除了给定图的m条边的信息，还会给定一个哈密顿回路（包含所有顶点）。

首先根据平面图性质（离散数学的图论部分有提到）m(图的边数)<=3*n-6(n是顶点数). 这样对于所有m>3n-6的情况直接输出“NO”，m的范围也缩小到600以内。

根据给定的哈密顿回路，可以确定所有顶点的顺序。

如图，就是样例种的哈密顿回路，绿色代表顶点顺序，而这个顶点顺序也是后面需要用到的。设id[i]表示顶点i出现的顺序是id[i] (以哈密顿回路中第一个出现的点为起点)

假如对于某两条边（都不在哈密顿回路上），

比如边(1,2)和边(4,5)显然顶点顺序满足 id[4]<id[2]<id[5].显然如果给定两条边(x,y)和(u,v)它们满足条件：id[u]<id[y]<id[v]&&id[x]<id[u]或者id[x]<id[v]<id[y]&&id[u]<id[x] ，这里必须保证两条边都不在哈密顿回路中，所以在一开始需要将哈密顿回路上的边全部标记出来即 sign[c[i]][c[i+1]]=sign[[c[i+1]][c[i]]=1，然后再枚举所有不在回路上的边

两条边必须在异侧，否则的话，两条边一定会相交（对于不等式中取等的情况，讨论分析后发现都不会出现相交，不必过多考虑，因为一定会在重合点上相交，但是不影响） 。假设两条边分别是i,j，边i，j表示边在图的内侧，边i+m，j+m表示边在图的外侧。对于这种情况建图一定是(i,j+m),(i+m,i)（无向图），其余不满足上述条件的两边不必考虑，因为一定不会相交，我们只需要考虑这些会彼此相交的成对边。

建图完成后，直接粘贴tarjan缩点模板，然后跑tarjan，最后判断是否所有的color[i]和color[i+m]都不在一个连通块中，如果存在color[i]=color[i+m]那就不是平面图。（一条边不可能同时在图的内侧又在图的外侧，所以要成为平面图color[i]一定不可以与color[i+m]相等）

        do{

int main(){
    cin>>T;
    while(T--){
        k=1;tot=0;
        memset(head,0,sizeof(head));
        memset(dfn,0,sizeof(dfn));
        memset(low,0,sizeof(low));
        memset(sta,0,sizeof(sta));
        memset(Num,0,sizeof(Num));
        memset(vis,0,sizeof(vis));
        memset(color,0,sizeof(color));

        cin>>n>>m;
        for(int i=1;i<=m;i++)cin>>a[i]>>b[i];
        for(int i=1;i<=n;i++){
            cin>>c[i];

        }
        if(m>3*n-6){
            cout<<"NO"<<endl;
            continue;
        }
        for(int i=1;i<n;i++)sign[c[i]][c[i+1]]=sign[c[i+1]][c[i]]=1;

        for(int i=1;i<=m;i++){
            if(!sign[a[i]][b[i]]){
                s.push_back(i);
            }
        }
        for(int i=0;i<s.size();i++){
            for(int j=i+1;j<s.size();j++){
                int x=id[a[s[i]]],y=id[b[s[i]]];

                if(x>y)swap(x,y);
                if(u>v)swap(u,v);
                if((u<x&&x<v&&v<y)||(x<u&&u<y&&y<v)){
                    add(i,j+m);add(j+m,i);
                    add(j,i+m);add(i+m,j);
                }
            }
        }
        for(int i=0;i<2*s.size();i++){
            if(!dfn[i]){
                tarjan(i);
            }
        }
        int sign=0;
        for(int i=0;i<s.size();i++){
            if(color[i]==color[i+m]){
                sign=1;
                break;
            }
        }
        if(sign)cout<<"NO"<<endl;
        else cout<<"YES"<<endl;
    }
}

3.洛谷P4782 【模板】2-SAT 问题

这道题相对于前面两个题的区别在于需要求解具体的方案（即赋值）

思路

这道题本身2-SAT建图很简单：xi=a|xj=b，如果xi=0，那么将让u=i+n；否则u=i；如果xj=0，那么就让v=j+n；否则v=j；xi=a|xj=b就转换成了u|v=1，即(~ u)->v,(~ v)->u.

2-SAT的输出方案：

在scc缩点之后的DAG图上，取一些点，具备以下条件：

1）对于所有变量x[i], scc[i]和scc[~ i]必选其一
2）若取scc[i],那么它的后代都要选 等价于若不选scc[i]，那么它的前代都不选。

由于原命题和逆否命题等价，所以缩点后的DAG具备对称性：scc[i]的后代与scc[~i]的前代对称

即a->b->c <==> (~ a)<-(~ b)<-(~ c)

1.选择scc[i]之后，scc[i]的所有后代必选；而scc[~ i]和scc[i]只能二选一，所以选了scc[i]就不能选scc[~ i]，并且scc[~ i]的所有前代也不能选
2.每次找到一个未选的scc[i]，保证不存在scc[i]->scc[~ i]，即scc[i]!=scc[i+n]（有解的充要条件）。选择scc[i]及其所有后代并且删除scc[~ i]及其所有前代。
3.用拓扑排序完成上述过程，每次选择出度为0的scc[i]，scc[i]没有后代 ,scc[~ i] 没有前代.

有Tarjan算法的性质可以发现：Tarjan求处的就是拓扑排序，它先求出了scc[i]小的点，而scc[i]最小的点同时也是拓扑序最大的。

如果scc[i]scc[~ i]，则说明i+n的拓扑序大，选择(~ i)，即x[i]=0;

        }
    }

        }
    }

        if(color[i]<color[i+n])cout<<1<<' ';//SCC序越小，表示拓扑序越大，首先选择拓扑序大的,即SCC序小的 
        else cout<<0<<' ';
    }
    return 0;
}

4.AcWing 404.婚礼

思路

假定：

i(xi=1)：妻子在新娘同侧，丈夫在新郎同侧；
~i(即xi=0)：妻子在新娘异侧，丈夫在新郎异侧；

根据上述假定，并且设新娘新郎对应x1，所以x1=1，因此(~ i)->i。

形如“4h 2w”表示第4对夫妇中的丈夫和第2对夫妇中的妻子通奸，那么就说明h4和w2不可以同时出现在新郎侧，而h4出现在新郎侧就等价于x4=1，w2出现在新郎侧就等价于x2=0，所以h4和w2不可以同时出现在新郎侧就等价于x4&(~ x2)=0，即x4->~ (~ x2),(~ x2)->x4

所以对于形如”xiyi xjyj”的说明xiyi和xjyj不可以同时出现在新郎侧，如果yi=”h”，那么xi出现在新郎侧就等价于xi=1，即u=i；如果yi=”h”，那么xi出现在新郎侧就等价于xi=0，即u=i+n;同理于xj，求出v。xiyi和xjyj不可以同时出现在新郎侧就等价于u&v=0,即u->(~ v),v->(~ u).

上述假定丈夫妻子的编号从2开始（因为新郎新娘为1）

接下来就是直接跑Tarjan套模板

输出的时候如果color[i]<color[i+n]就是指xi=1,即妻子和新娘同侧（注意i要减一（因为丈夫妻子的编号对比输入做了改变））否则就是指丈夫和新娘同侧。

5. AcWing 371. 牧师约翰最忙碌的一天

思路

每一个婚礼的主持时间有两种：t[i][0]=s[i],t[i][1]=t[i]-d[i]，这里的s[i]和t[i]需要将string转换成分钟数，方便操作

要求对于任意两个婚礼i，j要满足t[i][x]和t[j][y]不发生冲突，如果t[i][x]+d[i]<=t[j][y]|t[i][x]>=t[j][y]+d[j]表明不发生冲突。

假定i表示在婚礼的主持在t[i][0]开始，(~ i)即i+n表示在t[i][1]开始。

如果t[i][0]和t[j][0] 发生冲突：i->j+n,j+n->i;
如果t[i][0]和t[j][1] 发生冲突：i->j,j+n->i+n;
如果t[i][1]和t[j][0] 发生冲突：i+n->j+n,j->i;
如果t[i][1]和t[j][1] 发生冲突：i+n->j,j+n->i;

剩下的套模板即可，输出方案的时候记得分钟数转换成string。

    string s;
    int tot=x/60;
    s+=tot/10+'0';
    s+=tot%10+'0';
    x=x-tot*60;
    s+=':';
    s+=x/10+'0';
    s+=x%10+'0';
    return s;
}
int check(int i,int ci,int j,int cj){
    if(t[i][ci]+d[i]<=t[j][cj])return 0;
    if(t[i][ci]>=t[j][cj]+d[j])return 0;
    return 1;
}

        t[i][1]=str_to_int(y)-d[i];
    }
    for(int i=1;i<=n;i++){
        for(int j=1;j<=n;j++){
            if(i==j)continue;
            if(check(i,0,j,0))add(i,j+n),add(j,i+n);
            if(check(i,0,j,1))add(i,j),add(j+n,i+n);
            if(check(i,1,j,0))add(i+n,j+n),add(j,i);
            if(check(i,1,j,1))add(i+n,j),add(j+n,i);
        }
    }

    }
